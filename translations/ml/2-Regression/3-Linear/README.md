เดฒเตเดจเดฟเดฏเตผ เดฑเดเตเดฐเดทเตป เดฎเตเดกเตฝ เดชเดฐเดฟเดถเตเดฒเดฟเดชเตเดชเดฟเดเตเดเดพเตป เดจเดฎเตเดฎเตพ **Scikit-learn** เดฒเตเดฌเตเดฐเดฑเดฟ เดเดชเดฏเตเดเดฟเดเตเดเตเด.

เดจเดพเด เดเดฆเตเดฏเด เดเตปเดชเตเดเตเดเต เดฎเตเดฒเตเดฏเดเตเดเตพ (เดซเตเดเตเดเดฑเตเดเตพ) เดเดคเต เดธเดฎเดฏเด เดชเตเดฐเดคเตเดเตเดทเดฟเดเตเดเตเดจเตเดจ เดเดเตเดเตโเดชเตเดเตเดเต (เดฒเตเดฌเตฝ) numpy เดเดฐเตเดฏเดฟเดฒเดพเดฏเดฟ เดตเตเตผเดคเดฟเดฐเดฟเดเตเดเตเดจเตเดจเต:


> เดเตปเดชเตเดเตเดเต เดกเดพเดฑเตเดฑ เดชเดพเดเตเดเตเดเต เดถเดฐเดฟเดฏเดพเดฏเดฟ เดฎเดจเดธเตเดธเดฟเดฒเดพเดเตเดเดพเตป `reshape` เดจเดเดคเตเดคเตเดฃเตเดเดคเตเดฃเตเดเตเดจเตเดจเต เดถเตเดฐเดฆเตเดงเดฟเดเตเดเตเด. เดฒเตเดจเดฟเดฏเตผ เดฑเดเตเดฐเดทเตป 2D เดเดฐเต เดเดฃเตเดเต เดเตเตพเดเตเดเดพเตป เดเดเตเดฐเดนเดฟเดเตเดเตเดจเตเดจเต, เดเดฐเต เดฑเตเดพ เดกเดพเดฑเตเดฑเดพ เดธเดเตเดเตเดตเดเตเดเดณเตเดณเตเดณ เดซเตเดเตเดเดฑเตเดเดณเตเดเต เดตเตเดเตเดฑเตเดฑเดฑเดพเดฃเต. เดเดจเตเดจเดคเตเดคเต เดเดตเดธเดฐเดคเตเดคเดฟเตฝ, เดจเดฎเตเดฎเตเดเต เดเตปเดชเตเดเตเดเต เดเดฑเตเดฑ เดซเตเดเตเดเดฑเดฟเดจเตเดฏเดพเดฃเต, เดเดคเดฟเดจเดพเตฝ Nร1 เดฐเตเดชเดคเตเดคเดฟเดฒเตเดณเตเดณ เดเดฐเตเดฏเตเดฃเตเดเต, N dataset เดตเดฒเดฟเดชเตเดชเด.

เดเดคเดฟเดจเตเดถเตเดทเด เดกเดพเดฑเตเดฑ เดชเดฐเดฟเดถเตเดฒเดจเดคเตเดคเดฟเดจเตเด เดชเดฐเดฟเดถเตเดงเดจเดฏเตเดเตเดเตเดฎเดพเดฏเดฟ train-test เดธเตเดฑเตเดฑเตเดเดณเดพเดฏเดฟ เดตเตเตผเดคเดฟเดฐเดฟเดเตเดเตเด:

เดเดตเดธเดพเดจเดคเตเดคเดฟเตฝ, เดฏเดฅเดพเตผเดคเตเดฅ เดฒเตเดจเดฟเดฏเตผ เดฑเดเตเดฐเดทเตป เดฎเตเดกเตฝ เดชเดฐเดฟเดถเตเดฒเดฟเดชเตเดชเดฟเดเตเดเตเดจเตเดจเดคเต เดฐเดฃเตเดเต เดเตเดกเต เดตเดฐเดฟเดเดณเดฟเตฝ เดฎเดพเดคเตเดฐเดฎเต เดเดเตเดเตเดเต. `LinearRegression` เดเดฌเตโเดเดเตเดเต เดจเดฟเตผเดตเตเดตเดเดฟเดเตเดเต, `fit` เดฎเตเดคเดกเดฟเดฒเตเดเต เดกเดพเดฑเตเดฑเดฏเดฟเตฝ เดเดคเดฟเดจเต เดจเดฟเดจเตเดจเตเด เดซเดฟเดฑเตเดฑเต เดเตเดฏเตเดฏเตเด:


`LinearRegression` เดเดฌเตเดฐเดเตเดฑเตเดฑเต `fit`-เดเตเดฏเตโเดคเดคเดฟเดจเต เดถเตเดทเด เดฑเตเดเตเดฐเดทเดจเตเดฑเต เดเดฒเตเดฒเดพ เดเตเดซเดฟเดทเตเดฏเดจเตเดฑเตเดเดณเตเด เดเดเดเตเดเดฟเดฏเดฟเดฐเดฟเดเตเดเตเด, เดเดต `.coef_` เดชเตเดฐเตเดชเตเดชเตผเดเตเดเดฟ เดตเดดเดฟ เดเดเตเดธเดธเต เดเตเดฏเตเดฏเดพเดตเตเดจเตเดจเดคเดพเดฃเต. เดจเดฎเตเดฎเตเดเต เดเตเดธเดฟเตฝ เดเดฐเต เดเตเดซเดฟเดทเตเดฏเดจเตเดฑเต เดฎเดพเดคเตเดฐเดฎเต เดเดฃเตเดเดพเดเต, เดเดคเต เดเดเดฆเตเดถเด `-0.017` เดเดฏเดฟเดฐเดฟเดเตเดเดฃเด. เดเดคเต เดเดคเดพเดฏเดคเต เดตเดฟเดฒเดเตพ เดธเดฎเดฏเดคเตเดคเดฟเดจเต เดเดจเตเดฌเดจเตเดงเดฟเดเตเดเต เดเตเดฑเดเตเดเตเดเตเดฑเดเตเดเต เดเตเดฑเดเตเดเตเดเตเดเดพเด, เดชเดเตเดทเต เดเดคเตเดฐเดฏเตเด เดเตเดเตเดคเดฒเดฒเตเดฒ, เดฆเดฟเดตเดธเด เดเดเดฆเตเดถเด 2 เดธเตเดจเตเดฑเตเดเตพเดเตเดเต เดเตเดฑเตเดฑเตเด. Y-เดเดเตเดธเดฟเดธเตเดฎเดพเดฏเตเดณเตเดณ เดฑเตเดเตเดฐเดทเดจเตเดฑเต เดเดจเตเดฑเตเดดเตโเดธเตเดเตเดทเตป เดชเตเดฏเดฟเดจเตเดฑเต `lin_reg.intercept_` เดเดชเดฏเตเดเดฟเดเตเดเต เดเดเตเดธเดธเต เดเตเดฏเตเดฏเดพเดฎเตเดจเตเดจเดพเดฃเต, เดเดคเต เดจเดฎเตเดฎเตเดเต เดเตเดธเดฟเตฝ เดเดเดฆเตเดถเด `21` เดเดฏเดฟเดฐเดฟเดเตเดเตเด, เดตเตผเดทเดคเตเดคเดฟเดจเตเดฑเต เดคเตเดเดเตเดเดคเตเดคเดฟเดฒเต เดตเดฟเดฒ เดธเตเดเดฟเดชเตเดชเดฟเดเตเดเตเดจเตเดจเต.

เดจเดฎเตเดฎเตเดเต เดฎเตเดกเตฝ เดเดคเตเดฐเดคเตเดคเตเดณเด เดเดเตเดตเดฑเตเดฑเตเดฑเต เดเดฃเตเดจเตเดจเต เดเดพเดฃเดพเตป, เดเตเดธเตเดฑเตเดฑเต เดกเดพเดฑเตเดฑเดพเดธเตเดฑเตเดฑเดฟเตฝ เดตเดฟเดฒเดเตพ เดชเตเดฐเดตเดเดฟเดเตเดเต, เดชเตเดฐเดคเตเดเตเดทเดฟเดเตเด เดฎเตเดฒเตเดฏเดเตเดเดณเตเดเต เดจเดฎเตเดฎเตเดเต เดชเตเดฐเดตเดเดจเดเตเดเตพ เดเดคเตเดฐเดคเตเดคเตเดณเด เดเดเตเดคเตเดคเดพเดฃเต เดเดจเตเดจเต เดเดณเดเตเดเดพเด. เดเดคเต เดเดฃเดฟเดเตเดเตเดตเดพเตป เดฎเตเตป เดธเตเดเตเดตเดฏเตผ เดเดฑเตผ (MSE) เดฎเตเดเตเดฐเดฟเดเต เดเดชเดฏเตเดเดฟเดเตเดเดพเด, เดเดคเต เดชเตเดฐเดคเตเดเตเดทเดฟเดเตเดเดฏเตเด เดชเตเดฐเดตเดเดฟเดเตเดเดคเตเด เดคเดฎเตเดฎเดฟเดฒเตเดณเตเดณ เดเดฒเตเดฒเดพ เดธเตเดเตเดตเดฏเตผ เดเตเดฏเตเดค เดตเตเดฏเดคเตเดฏเดพเดธเดเตเดเดณเตเดเต เดถเดฐเดพเดถเดฐเดฟเดฏเดพเดฃเต.

```python

```
  
เดจเดฎเตเดฎเตเดเต เดคเดเดฐเดพเดฑเต เดเดเดฆเตเดถเด 2 เดชเตเดฏเดฟเดจเตเดฑเตเดเตพเดเตเดเต เดเตเดฑเตเดฑเตเด เดเดฃเต, เดเดคเต ~17% เดเตเดฒเดตเดพเดฃเต. เดเดคเตเดฐ เดจเดฒเตเดฒเดคเดฒเตเดฒ. เดฎเตเดกเดฒเดฟเดจเตเดฑเต เดเตเดฃเดจเดฟเดฒเดตเดพเดฐเดคเตเดคเดฟเดจเตเดฑเต เดฎเดฑเตเดฑเตเดฐเต เดธเตเดเดฟเดเดฏเดพเดฃเต **เดฆเตเดฑเตเดฑเตผเดฎเดฟเดจเตเดทเตป เดเตเดซเดฟเดทเตเดฏเดจเตเดฑเต** (coefficient of determination), เดเดคเต เดเดเตเดเดจเต เดฒเดญเดฟเดเตเดเตเด:

