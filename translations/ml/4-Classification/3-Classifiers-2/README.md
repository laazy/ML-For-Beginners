## เดฒเดฟเดจเดฟเดฏเตผ SVC เดเตเดฒเดพเดธเดฟเดซเดฏเดฐเตโ

Support-Vector clustering (SVC) Support-Vector machines เดเตเดเตเดเดฌเดคเตเดคเดฟเดฒเต เดเดฐเต เดถเดพเดเดฏเดพเดฃเต (เดเดตเดฏเต เดเตเดฑเดฟเดเตเดเต เดคเดพเดดเต เดชเดเดฟเดเตเดเดพเด). เด เดฐเตเดคเดฟเดฏเดฟเตฝ, เดฒเตเดฌเดฒเตเดเดณเต เดเดเตเดเดจเต เดเตเดฒเดธเตเดฑเตเดฑเตผ เดเตเดฏเตเดฏเดพเดฎเตเดจเตเดจเต เดคเตเดฐเตเดฎเดพเดจเดฟเดเตเดเดพเดจเตเดณเตเดณ เดเดฐเต 'kernel' เดคเดฟเดฐเดเตเดเตเดเตเดเตเดเดพเด. 'C' เดเตเดฐเดฎเตเดเตเดเด เดชเดพเดฐเดพเดฎเตเดฑเตเดฑเดฑเดฟเดจเตเดฑเต เดธเตเดตเดพเดงเตเดจเด เดจเดฟเดฏเดจเตเดคเตเดฐเดฟเดเตเดเตเดจเตเดจ 'regularization' เดเด เดธเตเดเดฟเดชเตเดชเดฟเดเตเดเตเดจเตเดจเต. kernel เดตเดฟเดตเดฟเดง [เดคเดฐเด](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVC.html#sklearn.svm.SVC) เดเดณเตเดณเดคเดฟเตฝ เดจเดฟเดจเตเดจเดพเดฃเต; เดเดตเดฟเดเต 'linear' เดเดฏเดฟ เดตเดเตเดเต linear SVC เดเดชเดฏเตเดพเดเดฟเดเตเดเตเดจเตเดจเดคเต เดเดฑเดชเตเดชเดพเดเตเดเตเดจเตเดจเต. Probability เดจเดฟเดจเตเดฑเต เดฎเตเดจเตเดจเดฑเดฟเดฏเดฟเดชเตเดชเต 'false' เดเดฃเตเดเตเดเดฟเดฒเตเด, เดเดตเดฟเดเต 'true' เดเดฏเดฟ เดเตเดฐเดฎเตเดเดฐเดฟเดเตเดเดฟเดฐเดฟเดเตเดเตเดจเตเดจเต เดธเดพเดงเตเดฏเดคเดพเดจเดฟเดฐเตเดเตเดทเดฃเดเตเดเดณเตโ เดถเตเดเดฐเดฟเดเตเดเดพเตป. เดกเดพเดฑเตเดฑ เดถเดซเดฟเตพ เดเตเดฏเตเดฏเดพเตป random state '0' เดเดฏเดฟ เดจเดฟเดถเตเดเดฏเดฟเดเตเดเดฟเดเตเดเตเดฃเตเดเต.

### เดเดญเตเดฏเดพเดธเด - เดฒเดฟเดจเดฟเดฏเตผ SVC เดชเตเดฐเดฏเตเดเดจเด เดเตเดฏเตเดฏเตเด

เดเตเดฒเดพเดธเดฟเดซเดฏเดฑเตเดเดณเตเดเต เดเดฐเต เดจเดฟเดฐ เดธเดเดฐเดเดฟเดเตเดเตเด, เดเดเตเดเตพ เดจเดฟเดฐเตโเดฃเดฏเด เดเตเดฏเตเดฏเตเดฎเตเดชเตเตพ เดเดคเดฟเดฒเตเดเตเดเต เดคเตเดเดฐเตเดเตเดเดฟ เดเตเตผเดเตเดเตเดจเตเดจเต.

2. Linear SVC เดเดชเดฏเตเดเดฟเดเตเดเต เดจเดฟเดเตเดเดณเตเดเต เดฎเตเดกเดฒเตโ เดชเดฐเดฟเดถเตเดฒเดฟเดชเตเดชเดฟเดเตเดเต เดฑเดฟเดชเตเดชเตเดฐเตโเดเตเดเต เดเดเตเดเดเดฟเดเตเดเตเด:
