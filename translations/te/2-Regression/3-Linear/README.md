‡∞Ü‡∞™‡±à, ‡∞Æ‡∞®‡∞Ç ‡∞°‡±á‡∞ü‡∞æ‡∞®‡±Å ‡∞ü‡±ç‡∞∞‡±Ü‡∞Ø‡∞ø‡∞®‡±ç ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞ü‡±Ü‡∞∏‡±ç‡∞ü‡±ç ‡∞°‡±á‡∞ü‡∞æ‡∞∏‡±Ü‡∞ü‡±ç ‡∞≤‡±Å‡∞ó‡∞æ ‡∞µ‡∞ø‡∞≠‡∞ú‡∞ø‡∞Ç‡∞ö‡∞æ‡∞≤‡∞ø, ‡∞§‡∞¶‡±ç‡∞µ‡∞æ‡∞∞‡∞æ ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞§‡∞∞‡±ç‡∞µ‡∞æ‡∞§ ‡∞Æ‡∞®‡∞Ç ‡∞Æ‡±ã‡∞°‡∞≤‡±ç‚Äå‡∞®‡±Å ‡∞∏‡∞∞‡±à‡∞®‡∞¶‡∞®‡∞ø ‡∞®‡∞ø‡∞∞‡±ç‡∞ß‡∞æ‡∞∞‡∞ø‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞ó‡∞≤‡∞®‡±Å:

```python
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
```
  
‡∞ö‡∞ø‡∞µ‡∞∞‡∞ó‡∞æ, ‡∞≤‡±Ä‡∞®‡∞ø‡∞Ø‡∞∞‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞®‡±Å ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞á‡∞µ‡±ç‡∞µ‡∞°‡∞Ç ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞ï‡±ã‡∞°‡±ç ‡∞≤‡±à‡∞®‡±ç‡∞≤‡∞≤‡±ã‡∞®‡±á ‡∞ú‡∞∞‡±Å‡∞ó‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞Æ‡±á‡∞Æ‡±Å `LinearRegression` ‡∞Ü‡∞¨‡±ç‡∞ú‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞®‡±Å ‡∞®‡∞ø‡∞∞‡±ç‡∞µ‡∞ö‡∞ø‡∞Ç‡∞ö‡∞ø, `fit` ‡∞Æ‡±Ü‡∞•‡∞°‡±ç ‡∞¶‡±ç‡∞µ‡∞æ‡∞∞‡∞æ ‡∞Æ‡∞® ‡∞°‡±á‡∞ü‡∞æ‡∞ï‡±Å ‡∞Ö‡∞®‡±Å‡∞ó‡±Å‡∞£‡∞Ç‡∞ó‡∞æ ‡∞∞‡±Ç‡∞™‡±ä‡∞Ç‡∞¶‡∞ø‡∞∏‡±ç‡∞§‡∞æ‡∞Æ‡±Å:

```python
lin_reg = LinearRegression()
lin_reg.fit(X_train,y_train)
```

`fit` ‡∞ö‡±á‡∞∏‡∞ø‡∞® ‡∞§‡∞∞‡±Å‡∞µ‡∞æ‡∞§ `LinearRegression` ‡∞Ü‡∞¨‡±ç‡∞ú‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï ‡∞Ö‡∞®‡±ç‡∞®‡∞ø ‡∞∏‡∞π‡∞ó‡±Å‡∞£‡∞æ‡∞≤‡∞®‡±Å ‡∞ï‡∞≤‡∞ø‡∞ó‡∞ø ‡∞â‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞µ‡∞æ‡∞ü‡∞ø‡∞®‡∞ø `.coef_` ‡∞™‡±ç‡∞∞‡∞æ‡∞™‡∞∞‡±ç‡∞ü‡±Ä ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞Ø‡∞æ‡∞ï‡±ç‡∞∏‡±Ü‡∞∏‡±ç ‡∞ö‡±á‡∞Ø‡∞µ‡∞ö‡±ç‡∞ö‡±Å. ‡∞Æ‡∞® ‡∞ï‡±á‡∞∏‡±Å‡∞≤‡±ã, ‡∞í‡∞ï‡±á ‡∞í‡∞ï‡±ç‡∞ï ‡∞∏‡∞π‡∞ó‡±Å‡∞£‡∞Ç ‡∞â‡∞Ç‡∞¶‡∞ø, ‡∞Ö‡∞¶‡∞ø ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å `-0.017` ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø. ‡∞¶‡±Ä‡∞®‡∞ø ‡∞Ö‡∞∞‡±ç‡∞•‡∞Ç ‡∞è‡∞Æ‡∞ø‡∞ü‡∞Ç‡∞ü‡±á, ‡∞ß‡∞∞‡∞≤‡±Å ‡∞ï‡∞æ‡∞≤‡∞ï‡±ç‡∞∞‡∞Æ‡±á‡∞£‡∞æ ‡∞ï‡±ä‡∞Ç‡∞§ ‡∞ï‡±ç‡∞∑‡±Ä‡∞£‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞®‡±ç‡∞®‡∞ü‡±ç‡∞≤‡±Å ‡∞ï‡∞®‡∞ø‡∞™‡∞ø‡∞∏‡±ç‡∞§‡±ã‡∞Ç‡∞¶‡∞ø, ‡∞∞‡±ã‡∞ú‡±Å‡∞ï‡±Å ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å 2 ‡∞∏‡±Ü‡∞Ç‡∞ü‡±ç‡∞≤‡±Å ‡∞§‡∞ó‡±ç‡∞ó‡±Å‡∞§‡±Å‡∞Ç‡∞ü‡∞æ‡∞Ø‡∞ø. ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï Y-‡∞Ö‡∞ï‡±ç‡∞∑‡∞§‡±ã ‡∞ï‡±Ç‡∞°‡∞ø‡∞® ‡∞á‡∞Ç‡∞ü‡∞∞‡±ç‚Äå‡∞∏‡±Ü‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡±ç ‡∞®‡±Å ‡∞Æ‡∞®‡∞Ç `lin_reg.intercept_` ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞ï‡±Ç‡∞°‡∞æ ‡∞Ø‡∞æ‡∞ï‡±ç‡∞∏‡±Ü‡∞∏‡±ç ‡∞ö‡±á‡∞Ø‡∞µ‡∞ö‡±ç‡∞ö‡±Å - ‡∞Æ‡∞® ‡∞ï‡±á‡∞∏‡±Å‡∞≤‡±ã ‡∞á‡∞¶‡∞ø ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å `21` ‡∞â‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞á‡∞¶‡∞ø ‡∞∏‡∞Ç‡∞µ‡∞§‡±ç‡∞∏‡∞∞‡∞Ç ‡∞™‡±ç‡∞∞‡∞æ‡∞∞‡∞Ç‡∞≠‡∞Ç‡∞≤‡±ã ‡∞ß‡∞∞‡∞®‡±Å ‡∞∏‡±Ç‡∞ö‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.

‡∞Æ‡∞® ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞é‡∞Ç‡∞§ ‡∞ñ‡∞ö‡±ç‡∞ö‡∞ø‡∞§‡∞Æ‡±ã ‡∞ö‡±Ç‡∞∏‡±á‡∞Ç‡∞¶‡±Å‡∞ï‡±Å, ‡∞Æ‡∞®‡∞Ç ‡∞ü‡±Ü‡∞∏‡±ç‡∞ü‡±ç ‡∞°‡±á‡∞ü‡∞æ‡∞∏‡±Ü‡∞ü‡±ç ‡∞™‡±à ‡∞ß‡∞∞‡∞≤‡∞®‡±Å ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ ‡∞µ‡±á‡∞∏‡∞ø, ‡∞Ö‡∞¶‡∞ø ‡∞Ü‡∞∂‡∞ø‡∞Ç‡∞ö‡∞ø‡∞® ‡∞Æ‡±Ç‡∞≤‡±ç‡∞Ø‡∞æ‡∞≤‡∞ï‡±Å ‡∞é‡∞Ç‡∞§ ‡∞¶‡∞ó‡±ç‡∞ó‡∞∞‡∞ó‡∞æ ‡∞â‡∞®‡±ç‡∞®‡∞¶‡±ã ‡∞ï‡±ä‡∞≤‡∞µ‡∞µ‡∞ö‡±ç‡∞ö‡±Å. ‡∞á‡∞¶‡∞ø mean square error (MSE) ‡∞™‡±ç‡∞∞‡∞Æ‡∞æ‡∞£‡∞Ç ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞ö‡±á‡∞Ø‡∞µ‡∞ö‡±ç‡∞ö‡±Å, ‡∞á‡∞¶‡∞ø ‡∞Ü‡∞∂‡∞ø‡∞Ç‡∞ö‡∞ø‡∞® ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ ‡∞µ‡∞ø‡∞≤‡±Å‡∞µ‡∞≤ ‡∞Æ‡∞ß‡±ç‡∞Ø ‡∞â‡∞®‡±ç‡∞® ‡∞Ö‡∞®‡±ç‡∞®‡∞ø ‡∞µ‡∞∞‡±ç‡∞ó ‡∞â‡∞®‡±ç‡∞®‡∞§ ‡∞µ‡±ç‡∞Ø‡∞§‡±ç‡∞Ø‡∞æ‡∞∏‡∞æ‡∞≤ ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï ‡∞∏‡∞ó‡∞ü‡±Å.


‡∞Æ‡∞® ‡∞§‡∞™‡±ç‡∞™‡∞ø‡∞¶‡∞Ç ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å 2 ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡±ç‡∞≤ ‡∞ö‡±Å‡∞ü‡±ç‡∞ü‡±Ç ‡∞â‡∞Ç‡∞¶‡∞ø, ‡∞á‡∞¶‡∞ø ~17%. ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞Æ‡∞Ç‡∞ö‡∞ø ‡∞ï‡∞æ‡∞¶‡±Å. ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞®‡∞æ‡∞£‡±ç‡∞Ø‡∞§‡∞ï‡±Å ‡∞Æ‡∞∞‡±ä‡∞ï ‡∞∏‡±Ç‡∞ö‡∞ø‡∞ï **coefficient of determination** ‡∞Ö‡∞®‡∞ø ‡∞™‡∞ø‡∞≤‡∞µ‡∞¨‡∞°‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞¶‡∞æ‡∞®‡±ç‡∞®‡∞ø ‡∞á‡∞≤‡∞æ ‡∞™‡±ä‡∞Ç‡∞¶‡∞µ‡∞ö‡±ç‡∞ö‡±Å:


‡∞Æ‡∞ø‡∞ó‡∞ø‡∞≤‡∞ø‡∞® ‡∞ï‡±ã‡∞°‡±ç ‡∞ï‡±Ç‡∞°‡∞æ ‡∞Æ‡∞®‡∞Ç ‡∞≤‡±Ä‡∞®‡∞ø‡∞Ø‡∞∞‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞ü‡±ç‡∞∞‡±Ü‡∞Ø‡∞ø‡∞®‡±ç ‡∞ö‡±á‡∞∏‡∞ø‡∞® ‡∞µ‡∞ø‡∞ß‡∞Ç‡∞ó‡∞æ ‡∞Ö‡∞¶‡±á. ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞™‡±ç‡∞∞‡∞Ø‡∞§‡±ç‡∞®‡∞ø‡∞∏‡±ç‡∞§‡±á, ‡∞Æ‡±Ä‡∞ï‡±Å ‡∞¶‡±á ‡∞ß‡∞æ‡∞∏‡∞ø‡∞® ‡∞Æ‡∞æ‡∞§‡±ç‡∞∞‡∞Ç ‡∞∏‡∞ó‡∞ü‡±Å ‡∞µ‡∞∞‡±ç‡∞ó‡∞Ç‡∞≤‡±ã ‡∞ï‡±Ç‡∞°‡∞ü‡∞™‡±Å ‡∞§‡∞™‡±ç‡∞™‡∞ø‡∞¶‡∞Ç (MSE) ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å ‡∞Ö‡∞¶‡±á ‡∞â‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞ï‡∞æ‡∞®‡±Ä determination coefficient 77% ‡∞µ‡∞∞‡∞ï‡±Å ‡∞™‡±Ü‡∞∞‡±Å‡∞ó‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞á‡∞Ç‡∞ï‡∞æ ‡∞ñ‡∞ö‡±ç‡∞ö‡∞ø‡∞§‡∞Æ‡±à‡∞® ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ‡∞≤‡±Å ‡∞™‡±ä‡∞Ç‡∞¶‡∞ü‡∞æ‡∞®‡∞ø‡∞ï‡∞ø, ‡∞Æ‡±á‡∞Æ‡±Å ‡∞Æ‡∞∞‡∞ø‡∞Ç‡∞§ categorical ‡∞´‡±Ä‡∞ö‡∞∞‡±ç‡∞≤‡±Å, ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø‡∞æ‡∞™‡∞∞‡∞Æ‡±à‡∞® ‡∞´‡±Ä‡∞ö‡∞∞‡±ç‡∞≤‡±Å, ‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£‡∞ï‡±Å `Month` ‡∞≤‡±á‡∞¶‡∞æ `DayOfYear` ‡∞®‡±Å ‡∞™‡∞∞‡∞ø‡∞ó‡∞£‡∞ø‡∞Ç‡∞ö‡∞µ‡∞ö‡±ç‡∞ö‡±Å. ‡∞Æ‡±ä‡∞§‡±ç‡∞§‡∞Ç ‡∞´‡±Ä‡∞ö‡∞∞‡±ç‡∞≤‡∞®‡±Å ‡∞í‡∞ï ‡∞™‡±Ü‡∞¶‡±ç‡∞¶ ‡∞Ö‡∞∞‡±ç‡∞∞‡±á ‡∞ó‡∞æ ‡∞ö‡±á‡∞Ø‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø `join` ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞µ‡∞ö‡±ç‡∞ö‡±Å:

## ‡∞Ö‡∞®‡±ç‡∞®‡∞ø‡∞ü‡∞ø‡∞®‡∞ø ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø‡∞§‡±á

‡∞Æ‡∞Ç‡∞ö‡∞ø ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞®‡±Å ‡∞∏‡±É‡∞∑‡±ç‡∞ü‡∞ø‡∞Ç‡∞ö‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø, ‡∞Æ‡∞®‡∞Ç ‡∞ï‡∞æ‡∞Ç‡∞¨‡∞ø‡∞®‡±ç‡∞°‡±ç (‡∞µ‡∞®‡±ç-‡∞π‡∞æ‡∞ü‡±ç ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡±ç ‡∞ö‡±á‡∞∏‡∞ø‡∞® categorical + ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø‡∞æ‡∞™‡∞∞‡∞Æ‡±à‡∞®) ‡∞°‡±á‡∞ü‡∞æ‡∞®‡±Å ‡∞™‡±à ‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£ ‡∞®‡±Å‡∞Ç‡∞°‡∞ø ‡∞™‡±ã‡∞≤‡∞ø‡∞®‡±ã‡∞Æ‡∞ø‡∞Ø‡∞≤‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞§‡±ã ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø ‡∞µ‡∞æ‡∞°‡∞µ‡∞ö‡±ç‡∞ö‡±Å. ‡∞Æ‡±Ä ‡∞∏‡±å‡∞≤‡∞≠‡±ç‡∞Ø‡∞Ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞™‡±Ç‡∞∞‡±ç‡∞§‡∞ø ‡∞ï‡±ã‡∞°‡±ç ‡∞á‡∞ï‡±ç‡∞ï‡∞° ‡∞â‡∞Ç‡∞¶‡∞ø:

```python
# ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞°‡±á‡∞ü‡∞æ‡∞®‡±Å ‡∞∏‡±Ü‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
X = pd.get_dummies(new_pumpkins['Variety']) \
        .join(new_pumpkins['Month']) \
        .join(pd.get_dummies(new_pumpkins['City'])) \
        .join(pd.get_dummies(new_pumpkins['Package']))

# ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£-‡∞™‡∞∞‡±Ä‡∞ï‡±ç‡∞∑ ‡∞µ‡∞ø‡∞°‡∞ó‡±ä‡∞ü‡±ç‡∞ü‡±Å ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# ‡∞™‡±à‡∞™‡±ç‚Äå‡∞≤‡±à‡∞®‡±ç‚Äå‡∞®‡±Å ‡∞∏‡±Ü‡∞ü‡∞™‡±ç ‡∞ö‡±á‡∞∏‡∞ø ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞á‡∞µ‡±ç‡∞µ‡∞Ç‡∞°‡∞ø
pipeline = make_pipeline(PolynomialFeatures(2), LinearRegression())

# ‡∞™‡∞∞‡±Ä‡∞ï‡±ç‡∞∑ ‡∞°‡±á‡∞ü‡∞æ ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞´‡∞≤‡∞ø‡∞§‡∞æ‡∞≤‡±Å ‡∞≠‡∞µ‡∞ø‡∞∑‡±ç‡∞Ø‡∞§‡±ç‡∞§‡±Å ‡∞ö‡±Ü‡∞™‡±ç‡∞™‡∞Ç‡∞°‡∞ø
pred = pipeline.predict(X_test)

# MSE ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞®‡∞ø‡∞∞‡±ç‡∞£‡∞Ø‡∞Ç ‡∞ó‡∞£‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø
mse = np.sqrt(mean_squared_error(y_test,pred))
print(f'Mean error: {mse:3.3} ({mse/np.mean(pred)*100:3.3}%)')

score = pipeline.score(X_train,y_train)
```
