> ‡≤ó‡≤Æ‡≤®‡≤ø‡≤∏‡≤ø, ‡≤®‡≤æ‡≤µ‡≥Å ‡≤á‡≤®‡≥ç‡≤™‡≥Å‡≤ü‡≥ç ‡≤°‡≥á‡≤ü‡≤æ‡≤µ‡≤®‡≥ç‡≤®‡≥Å ‡≤∏‡≤∞‡≤ø‡≤Ø‡≤æ‡≤ó‡≤ø ‡≤§‡≤ú‡≥ç‡≤û‡≤∞‡≤ø‡≤§‡≤ø‡≤∏‡≤≤‡≥Å `reshape` ‡≤Æ‡≤æ‡≤°‡≤¨‡≥á‡≤ï‡≤æ‡≤Ø‡≤ø‡≤§‡≥Å. ‡≤≤‡≤ø‡≤®‡≤ø‡≤Ø‡≤∞‡≥ç ‡≤∞‡≥Ü‡≤ó‡≥ç‡≤∞‡≥Ü‡≤∑‡≤®‡≥ç 2D ‡≤Ö‡≤∞‡≥Ü ‡≤á‡≤®‡≥ç‡≤™‡≥Å‡≤ü‡≥ç ‡≤Ö‡≤®‡≥ç‡≤®‡≥Å ‡≤®‡≤ø‡≤∞‡≥Ä‡≤ï‡≥ç‡≤∑‡≤ø‡≤∏‡≥Å‡≤§‡≥ç‡≤§‡≤¶‡≥Ü, ‡≤Ö‡≤≤‡≥ç‡≤≤‡≤ø ‡≤™‡≥ç‡≤∞‡≤§‡≤ø‡≤Ø‡≥ä‡≤Ç‡≤¶‡≥Å ‡≤∏‡≤æ‡≤≤‡≥Å ‡≤µ‡≥à‡≤∂‡≤ø‡≤∑‡≥ç‡≤ü‡≥ç‡≤Ø‡≤ó‡≤≥ ‡≤í‡≤Ç‡≤¶‡≥Å ‡≤µ‡≤ø‡≤ï‡≥ç‡≤ü‡≤∞‡≥ç ‡≤Ü‡≤ó‡≤ø‡≤∞‡≥Å‡≤§‡≥ç‡≤§‡≤¶‡≥Ü. ‡≤®‡≤Æ‡≥ç‡≤Æ ‡≤™‡≥ç‡≤∞‡≤ï‡≤∞‡≤£‡≤¶‡≤≤‡≥ç‡≤≤‡≤ø ‡≤í‡≤Ç‡≤¶‡≥á ‡≤á‡≤®‡≥ç‡≤™‡≥Å‡≤ü‡≥ç ‡≤á‡≤¶‡≥ç‡≤¶‡≥Å, N√ó1 ‡≤Ö‡≤∞‡≥Ü ‡≤¨‡≥á‡≤ï‡≤æ‡≤ó‡≤ø‡≤¶‡≥Ü, ‡≤á‡≤≤‡≥ç‡≤≤‡≤ø N ‡≤°‡≥á‡≤ü‡≤æ‡≤∏‡≥Ü‡≤ü‡≥ç ‡≤ó‡≤æ‡≤§‡≥ç‡≤∞.

`fit` ‡≤Æ‡≤æ‡≤°‡≥Å‡≤µ ‡≤®‡≤Ç‡≤§‡≤∞‡≤¶ `LinearRegression` ‡≤µ‡≤∏‡≥ç‡≤§‡≥Å‡≤µ‡≥Å regression ‡≤® ‡≤é‡≤≤‡≥ç‡≤≤‡≤æ coefficients ‡≤ó‡≤≥‡≤®‡≥ç‡≤®‡≥Å ‡≤π‡≥ä‡≤Ç‡≤¶‡≤ø‡≤∞‡≥Å‡≤§‡≥ç‡≤§‡≤¶‡≥Ü, ‡≤á‡≤¶‡≤ï‡≥ç‡≤ï‡≥Ü `.coef_` ‡≤é‡≤Ç‡≤¨ property ‡≤â‡≤™‡≤Ø‡≥ã‡≤ó‡≤ø‡≤∏‡≤ø ‡≤™‡≥ç‡≤∞‡≤µ‡≥á‡≤∂‡≤ø‡≤∏‡≤¨‡≤π‡≥Å‡≤¶‡≥Å. ‡≤®‡≤Æ‡≥ç‡≤Æ ‡≤™‡≥ç‡≤∞‡≤ï‡≤∞‡≤£‡≤¶‡≤≤‡≥ç‡≤≤‡≤ø, ‡≤ï‡≥á‡≤µ‡≤≤ ‡≤í‡≤Ç‡≤¶‡≥Å coefficient ‡≤á‡≤¶‡≥Ü, ‡≤Ö‡≤¶‡≥Å ‡≤∏‡±Å‡∞Æ‡≤æ‡≤∞‡≤ø‡≤ó‡≥Ü `-0.017` ‡≤Ü‡≤ó‡≤ø‡≤∞‡≤¨‡≤π‡≥Å‡≤¶‡≥Å. ‡≤á‡≤¶‡≤∞ ‡≤Ö‡≤∞‡≥ç‡≤• ‡≤¨‡≥Ü‡≤≤‡≥Ü‡≤ó‡≤≥‡≥Å ‡≤ï‡≤æ‡≤≤‡≤¶‡≥ä‡≤Ç‡≤¶‡≤ø‡≤ó‡≥Ü ‡≤∏‡≥ç‡≤µ‡≤≤‡≥ç‡≤™ ‡≤á‡≤≥‡≤ø‡≤ï‡≥Ü‡≤Ø‡≤æ‡≤ó‡≥Å‡≤§‡≥ç‡≤§‡≤ø‡≤µ‡≥Ü, ‡≤Ü‡≤¶‡≤∞‡≥Ü ‡≤¨‡≤π‡≤≥‡≤∑‡≥ç‡≤ü‡≥Å ‡≤Ö‡≤≤‡≥ç‡≤≤, ‡≤¶‡≤ø‡≤®‡≤ï‡≥ç‡≤ï‡≥Ü ‡≤∏‡≥Å‡≤Æ‡≤æ‡≤∞‡≥Å 2 ‡≤∏‡≥Ü‡≤Ç‡≤ü‡≥Å‡≤ó‡≤≥‡≥Å ‡≤á‡≤≥‡≤ø‡≤Ø‡≥Å‡≤§‡≥ç‡≤§‡≤µ‡≥Ü. regression ‡≤® Y-‡≤Ö‡≤ï‡≥ç‡≤∑‡≤¶ ‡≤®‡≤°‡≥Å‡≤µ‡≥Ü ‡≤á‡≤∞‡≥Å‡≤µ intersection point ‡≤Ö‡≤®‡≥ç‡≤®‡≥Å ‡≤®‡≤æ‡≤µ‡≥Å `lin_reg.intercept_` ‡≤â‡≤™‡≤Ø‡≥ã‡≤ó‡≤ø‡≤∏‡≤ø ‡≤™‡≥ç‡≤∞‡≤µ‡≥á‡≤∂‡≤ø‡≤∏‡≤¨‡≤π‡≥Å‡≤¶‡≥Å - ‡≤á‡≤¶‡≥Å ‡≤®‡≤Æ‡≥ç‡≤Æ ‡≤™‡≥ç‡≤∞‡≤ï‡≤∞‡≤£‡≤¶‡≤≤‡≥ç‡≤≤‡≤ø ‡≤∏‡±Å‡∞Æ‡≤æ‡≤∞‡≤ø‡≤ó‡≥Ü `21` ‡≤á‡≤∞‡≤¨‡≤π‡≥Å‡≤¶‡≥Å, ‡≤á‡≤¶‡≥Å ‡≤µ‡≤∞‡≥ç‡≤∑‡≤¶ ‡≤™‡≥ç‡≤∞‡≤æ‡≤∞‡≤Ç‡≤≠‡≤¶‡≤≤‡≥ç‡≤≤‡≤ø ‡≤¨‡≥Ü‡≤≤‡≥Ü‡≤Ø‡≤®‡≥ç‡≤®‡≥Å ‡≤∏‡≥Ç‡≤ö‡≤ø‡≤∏‡≥Å‡≤§‡≥ç‡≤§‡≤¶‡≥Ü.

‡≤®‡≤Æ‡≥ç‡≤Æ ‡≤Æ‡≤æ‡≤¶‡≤∞‡≤ø ‡≤é‡≤∑‡≥ç‡≤ü‡≥Å ‡≤ñ‡≤ö‡≤ø‡≤§‡≤µ‡≤æ‡≤ó‡≤ø‡≤¶‡≥Ü ‡≤é‡≤Ç‡≤¶‡≥Å ‡≤®‡≥ã‡≤°‡≤≤‡≥Å, ‡≤®‡≤æ‡≤µ‡≥Å ‡≤™‡≤∞‡≥Ä‡≤ï‡≥ç‡≤∑‡≤æ ‡≤°‡≥á‡≤ü‡≤æ‡≤∏‡≥Ü‡≤ü‡≥ç‚Äå‡≤®‡≤≤‡≥ç‡≤≤‡≤ø ‡≤¨‡≥Ü‡≤≤‡≥Ü‡≤Ø‡≤®‡≥ç‡≤®‡≥Å predict ‡≤Æ‡≤æ‡≤°‡≤¨‡≤π‡≥Å‡≤¶‡≥Å, ‡≤®‡≤Ç‡≤§‡≤∞ ‡≤®‡≤Æ‡≥ç‡≤Æ ‡≤≠‡≤µ‡≤ø‡≤∑‡≥ç‡≤Ø‡≤µ‡≤æ‡≤£‡≤ø ‡≤®‡≤ø‡≤∞‡≥Ä‡≤ï‡≥ç‡≤∑‡≤ø‡≤§ ‡≤Æ‡≥å‡≤≤‡≥ç‡≤Ø‡≤ó‡≤≥‡≤ø‡≤ó‡≥Ü ‡≤é‡≤∑‡≥ç‡≤ü‡≥Å ‡≤π‡≤§‡≥ç‡≤§‡≤ø‡≤∞‡≤µ‡≤æ‡≤ó‡≤ø‡≤¶‡≥Ü ‡≤é‡≤Ç‡≤¶‡≥Å ‡≤Ö‡≤≥‡≥Ü‡≤Ø‡≤¨‡≤π‡≥Å‡≤¶‡≥Å. ‡≤á‡≤¶‡≥Å mean square error (MSE) ‡≤Æ‡≥Ü‡≤ü‡≥ç‡≤∞‡≤ø‡≤ï‡≥ç‡≤∏‡≥ç ‡≤¨‡≤≥‡≤∏‡≤ø ‡≤Æ‡≤æ‡≤°‡≤¨‡≤π‡≥Å‡≤¶‡≥Å, ‡≤á‡≤¶‡≥Å ‡≤®‡≤ø‡≤∞‡≥Ä‡≤ï‡≥ç‡≤∑‡≤ø‡≤§ ‡≤Æ‡≤§‡≥ç‡≤§‡≥Å ‡≤≠‡≤µ‡≤ø‡≤∑‡≥ç‡≤Ø‡≤µ‡≤æ‡≤£‡≤ø ‡≤Æ‡≥å‡≤≤‡≥ç‡≤Ø‡≤¶ ‡≤®‡≤°‡≥Å‡≤µ‡≤ø‡≤® ‡≤é‡≤≤‡≥ç‡≤≤‡≤æ ‡≤µ‡≤∞‡≥ç‡≤ó‡≥Ä‡≤ï‡≤∞‡≤ø‡≤∏‡≤ø‡≤¶ ‡≤µ‡≥ç‡≤Ø‡≤§‡≥ç‡≤Ø‡≤æ‡≤∏‡≤ó‡≤≥ ‡≤∏‡≤∞‡≤æ‡≤∏‡≤∞‡≤ø.

## ‡≤é‡≤≤‡≥ç‡≤≤‡≤µ‡≤®‡≥ç‡≤®‡≥Ç ‡≤í‡≤ü‡≥ç‡≤ü‡≥Å‡≤ó‡≥Ç‡≤°‡≤ø‡≤∏‡≥Å‡≤µ‡≥Å‡≤¶‡≥Å

        .join(new_pumpkins['Month']) \

        .join(pd.get_dummies(new_pumpkins['Package']))

# ‡≤§‡≤∞‡≤¨‡≥á‡≤§‡≤ø-‡≤™‡≤∞‡≥Ä‡≤ï‡≥ç‡≤∑‡≥Ü ‡≤π‡≥ã‡≤≤‡≤ø‡≤ï‡≥Ü‡≤Ø‡≤®‡≥ç‡≤®‡≥Å ‡≤Æ‡≤æ‡≤°‡≥Å

# ‡≤™‡≤∞‡≥Ä‡≤ï‡≥ç‡≤∑‡≤æ ‡≤°‡≥á‡≤ü‡≤æ‡≤¶ ‡≤´‡≤≤‡≤ø‡≤§‡≤æ‡≤Ç‡≤∂‡≤ó‡≤≥‡≤®‡≥ç‡≤®‡≥Å ‡≤ä‡≤π‡≤ø‡≤∏‡≥Å
pred = pipeline.predict(X_test)
