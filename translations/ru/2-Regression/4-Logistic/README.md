### –ü—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω–∞—è –æ–±—Ä–∞–±–æ—Ç–∫–∞ –¥–∞–Ω–Ω—ã—Ö: –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ –∏ –º–µ—Ç–æ–∫

–ù–∞—à –Ω–∞–±–æ—Ä –¥–∞–Ω–Ω—ã—Ö –æ —Ç—ã–∫–≤–∞—Ö —Å–æ–¥–µ—Ä–∂–∏—Ç —Å—Ç—Ä–æ–∫–æ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –¥–ª—è –≤—Å–µ—Ö —Å–≤–æ–∏—Ö —Å—Ç–æ–ª–±—Ü–æ–≤. –†–∞–±–æ—Ç–∞ —Å –∫–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–º–∏ –¥–∞–Ω–Ω—ã–º–∏ –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ –ø–æ–Ω—è—Ç–Ω–∞ –¥–ª—è –ª—é–¥–µ–π, –Ω–æ –Ω–µ –¥–ª—è –º–∞—à–∏–Ω. –ê–ª–≥–æ—Ä–∏—Ç–º—ã –º–∞—à–∏–Ω–Ω–æ–≥–æ –æ–±—É—á–µ–Ω–∏—è —Ö–æ—Ä–æ—à–æ —Ä–∞–±–æ—Ç–∞—é—Ç —Å —á–∏—Å–ª–∞–º–∏. –ò–º–µ–Ω–Ω–æ –ø–æ—ç—Ç–æ–º—É –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ —è–≤–ª—è–µ—Ç—Å—è –æ—á–µ–Ω—å –≤–∞–∂–Ω—ã–º —ç—Ç–∞–ø–æ–º –Ω–∞ —ç—Ç–∞–ø–µ –ø—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω–æ–π –æ–±—Ä–∞–±–æ—Ç–∫–∏ –¥–∞–Ω–Ω—ã—Ö, —Ç–∞–∫ –∫–∞–∫ –æ–Ω–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –∫–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –≤ —á–∏—Å–ª–æ–≤—ã–µ, –Ω–µ —Ç–µ—Ä—è—è –ø—Ä–∏ —ç—Ç–æ–º –Ω–∏–∫–∞–∫–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏. –•–æ—Ä–æ—à–µ–µ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–ø–æ—Å–æ–±—Å—Ç–≤—É–µ—Ç —Å–æ–∑–¥–∞–Ω–∏—é —Ö–æ—Ä–æ—à–µ–π –º–æ–¥–µ–ª–∏.

–î–ª—è –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Ç–∏–ø–∞ –∫–æ–¥–∏—Ä–æ–≤—â–∏–∫–æ–≤:

1. Ordinal encoder: –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –ø–æ—Ä—è–¥–∫–æ–≤—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –∫–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏, –≥–¥–µ –∏—Ö –¥–∞–Ω–Ω—ã–µ —Å–ª–µ–¥—É—é—Ç –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É –ø–æ—Ä—è–¥–∫—É, –Ω–∞–ø—Ä–∏–º–µ—Ä, —Å—Ç–æ–ª–±–µ—Ü `Item Size` –≤ –Ω–∞—à–µ–º –Ω–∞–±–æ—Ä–µ –¥–∞–Ω–Ω—ã—Ö. –û–Ω —Å–æ–∑–¥–∞–µ—Ç –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ, –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º –∫–∞–∂–¥–∞—è –∫–∞—Ç–µ–≥–æ—Ä–∏—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ —á–∏—Å–ª–æ–º, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–º –ø–æ—Ä—è–¥–∫—É –∫–∞—Ç–µ–≥–æ—Ä–∏–∏ –≤ —Å—Ç–æ–ª–±—Ü–µ.

    ```python
    from sklearn.preprocessing import OrdinalEncoder

    item_size_categories = [['sml', 'med', 'med-lge', 'lge', 'xlge', 'jbo', 'exjbo']]
    ordinal_features = ['Item Size']
    ordinal_encoder = OrdinalEncoder(categories=item_size_categories)
    ```

2. Categorical encoder: –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –Ω–æ–º–∏–Ω–∞–ª—å–Ω—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –∫–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏, –≥–¥–µ –∏—Ö –¥–∞–Ω–Ω—ã–µ –Ω–µ —Å–ª–µ–¥—É—é—Ç –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É –ø–æ—Ä—è–¥–∫—É, –Ω–∞–ø—Ä–∏–º–µ—Ä, –≤—Å–µ –ø—Ä–∏–∑–Ω–∞–∫–∏, –æ—Ç–ª–∏—á–Ω—ã–µ –æ—Ç `Item Size` –≤ –Ω–∞—à–µ–º –Ω–∞–±–æ—Ä–µ –¥–∞–Ω–Ω—ã—Ö. –≠—Ç–æ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é "–æ–¥–Ω–æ–≥–æ –≥–æ—Ä—è—á–µ–≥–æ", —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –∫–∞–∂–¥–∞—è –∫–∞—Ç–µ–≥–æ—Ä–∏—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –±–∏–Ω–∞—Ä–Ω—ã–º —Å—Ç–æ–ª–±—Ü–æ–º: –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è —Ä–∞–≤–Ω–∞ 1, –µ—Å–ª–∏ —Ç—ã–∫–≤–∞ –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–∏—Ç —ç—Ç–æ–º—É —Å–æ—Ä—Ç—É, –∏ 0 –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ.

    ```python
    from sklearn.preprocessing import OneHotEncoder

    categorical_features = ['City Name', 'Package', 'Variety', 'Origin']
    categorical_encoder = OneHotEncoder(sparse_output=False)
    ```

–ó–∞—Ç–µ–º `ColumnTransformer` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∫–æ–¥–∏—Ä–æ–≤—â–∏–∫–æ–≤ –≤ –æ–¥–∏–Ω —à–∞–≥ –∏ –∏—Ö –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –∫ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–º —Å—Ç–æ–ª–±—Ü–∞–º.

```python
    from sklearn.compose import ColumnTransformer
    
    ct = ColumnTransformer(transformers=[
        ('ord', ordinal_encoder, ordinal_features),
        ('cat', categorical_encoder, categorical_features)
        ])
    
    ct.set_output(transform='pandas')
    encoded_features = ct.fit_transform(pumpkins)
```

–° –¥—Ä—É–≥–æ–π —Å—Ç–æ—Ä–æ–Ω—ã, –¥–ª—è –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è –º–µ—Ç–∫–∏ –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º –∫–ª–∞—Å—Å `LabelEncoder` –∏–∑ scikit-learn, –∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è —É—Ç–∏–ª–∏—Ç–æ–π –¥–ª—è –Ω–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏–∏ –º–µ—Ç–æ–∫, —á—Ç–æ–±—ã –æ–Ω–∏ —Å–æ–¥–µ—Ä–∂–∞–ª–∏ —Ç–æ–ª—å–∫–æ –∑–Ω–∞—á–µ–Ω–∏—è –º–µ–∂–¥—É 0 –∏ n_classes-1 (–∑–¥–µ—Å—å, 0 –∏ 1).

```python
    from sklearn.preprocessing import LabelEncoder

    label_encoder = LabelEncoder()
    encoded_label = label_encoder.fit_transform(pumpkins['Color'])
```

–ü–æ—Å–ª–µ —Ç–æ–≥–æ –∫–∞–∫ –º—ã –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–ª–∏ –ø—Ä–∏–∑–Ω–∞–∫–∏ –∏ –º–µ—Ç–∫—É, –º—ã –º–æ–∂–µ–º –æ–±—ä–µ–¥–∏–Ω–∏—Ç—å –∏—Ö –≤ –Ω–æ–≤—ã–π –¥–∞—Ç–∞—Ñ—Ä–µ–π–º `encoded_pumpkins`.

```python
    encoded_pumpkins = encoded_features.assign(Color=encoded_label)
```
